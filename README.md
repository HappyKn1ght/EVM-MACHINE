# 🗳️ EVM Machine

A simple Electronic Voting Machine simulation built in C++.
Features user verification, CAPTCHA system and UID validation.

## ⚡ Features

- User registration with name and UID
- 10-digit UID validation
- Random CAPTCHA verification
- Loading bar animation
- Clean UI with screen clearing
- Voting system (coming soon!)

## 📥 Installation

1. Clone the repository:
```bash
git clone https://github.com/HappyKn1ght/EVM-Machine.git
```

2. Navigate to project folder:
```bash
cd EVM-Machine
```

## 🚀 How to Compile & Run

```bash
g++ evm.cpp -o evm && ./evm
```
